KIKA is a mobile application designed to modernize the traditional African tontine system. It enables users to create, manage, and participate in savings circles with complete transparency and security.

## Key Features

- Digital tontine creation with customizable contribution amounts and frequencies
- Mobile Money integration (Orange, MTN, Wave) for seamless payments
- Automated reminders and payment tracking
- Role-based access control for tontine administrators
- Real-time notifications and activity feeds
- Secure fund distribution based on defined rotation order

## Technical Implementation

Built a cross-platform mobile application using React Native with a robust Node.js backend. Integrated multiple mobile money APIs for pan-African coverage. Implemented real-time updates using WebSockets and ensured data security with end-to-end encryption.
